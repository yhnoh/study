### 스터디 방식
- 모두가 동일한 주제를 1주에 1번씩 스터디 진행 (스터디 시작일: ?)
- 부득이하게 스터디날 스터디를 할 수 없게 될 경우 일정 조정
- 발표는 당일 제비뽑기를 통해서 발표자 뽑기
- 기본적으로 스터디 운영자는 발표 자료를 준비하지 않음
  - 발표 이후에 해당 내용을 정확히 이해하고 있는지 질의할 예정
  - 스터디 운영자도 잘 모르는 부분에 대해서는 발표를 할 예정

### 스터디 목적
- 자바에 대해 이해하고 있다는 착각을 해소하기 위한 스터디
- 자바가 제공하는 주요 인터페이스 및 프레임워크에 대한 이해
- 자바가 제공하는 I/O에 대한 이해
- 자바가 제공하는 멀티스레드에 대한 이해
